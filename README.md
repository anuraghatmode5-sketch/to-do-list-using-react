# ✅ React Todo App

React-based To-Do List app with task management, completion tracking, Local Storage persistence, and responsive UI.

## 📚 What I Learned
- ⚛️ useState, useRef
- 📦 Props & Component Architecture
- 🗂️ List Rendering with `.map()` & `.filter()`
- 🎨 Dynamic Styling & Conditional Rendering
- 💾 localStorage for data persistence

## ✨ Features
- ➕ Add Tasks
- ✅ Mark as Completed
- 🗑️ Delete Tasks
- ⚡ Dynamic UI Updates
